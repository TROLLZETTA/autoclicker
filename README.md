# AutoClicker Pro

Uma ferramenta de automação em Python com interface gráfica (GUI), projetada para cliques automatizados e pressionamento de teclas com alta precisão e facilidade de uso.

## 🚀 Funcionalidades

* **Interface Futurista/Suave:** Interface moderna desenvolvida com `Tkinter`.
* **Modos de Ação:** Alternância dinâmica entre cliques de mouse (com coordenadas customizáveis) ou pressionamento de teclas.
* **Captura Inteligente:** Modo de captura de coordenadas em tempo real através de cliques na tela.
* **Controle de Fluxo:** Opção de definir número exato de repetições ou rodar em loop infinito.
* **Segurança:** Tecla de parada de emergência (ESC) global.
* **Overlay:** Janela sempre no topo (`Always On Top`) para facilitar o uso sobre o navegador ou aplicações.

## 🛠️ Tecnologias Utilizadas

* **Python 3.13**
* **Tkinter:** Para a interface gráfica.
* **PyAutoGUI:** Para controle de mouse e teclado.
* **Pynput:** Para captura de eventos de clique.
* **Keyboard:** Para monitoramento de teclas de atalho globais.

## 📋 Pré-requisitos

Para rodar o script, você precisará ter o Python instalado e as bibliotecas listadas abaixo:


pip install pyautogui keyboard pynput
